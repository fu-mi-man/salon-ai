# ダッシュボード画面

**ルーティング**: `/`
**対象ユーザー**: オーナー
**主な目的**: 生成された返信文を確認・編集・コピーする

## 画面構成

```
ヘッダー
  └ サービス名

フィルタータブ
  └ すべて / 未対応 / 対応済み

返信文カード一覧
  └ 返信文カード × N

```

## フィルタータブ

| タブ | 表示対象 |
|---|---|
| すべて | 全ステータス |
| 未対応 | pending・in_progress |
| 対応済み | done |

デフォルトは「未対応」タブを選択した状態で表示する。

## 返信文カード

1件の返信文を1枚のカードで表示する。

### 表示項目

| 項目 | 内容 |
|---|---|
| 生成日時 | 例: 2026/04/03 08:00 |
| ステータスバッジ | pending →「未対応」/ in_progress →「確認中」/ done →「対応済み」|
| 返信文本文 | 生成されたテキスト全文 |
| トーン | 生成時のプリセット（丁寧 / カジュアル / 簡潔） |

### 操作

| 操作 | 挙動 |
|---|---|
| コピーボタン | 返信文をクリップボードにコピーする。コピー後はボタンのラベルが「コピーしました」に変わり2秒後に戻る |
| 対応済みにするボタン | ステータスをdoneに更新する。「未対応」タブ表示中はカードが一覧から消える |
| 再生成ボタン | トーン選択ドロワーを開く |

### 再生成ドロワー

「再生成」ボタンを押すと画面下からドロワーが開く。

```
トーンを選択してください

○ 丁寧（現在の設定）
○ カジュアル
○ 簡潔

[再生成する]ボタン
```

「再生成する」を押すとGemini APIを呼び出し、返信文を上書き保存する。
生成中はローディングスピナーを表示する。

## 状態

| 状態 | 表示 |
|---|---|
| 読み込み中 | カードのスケルトンローダーを表示する |
| 返信文が0件 | 「対応が必要な口コミはありません」と表示する |
| コピー成功 | ボタンラベルを「コピーしました」に変更する（2秒） |
| 再生成中 | ドロワー内のボタンをローディング状態にする |
| 再生成成功 | ドロワーを閉じ、カードの返信文を更新する |
| 再生成失敗 | ドロワー内にエラーメッセージを表示する |

## 備考

- SPファースト。カードは縦1列で表示する
- フェーズ1は認証なし。URLを知っていれば誰でもアクセスできる
- サロンボードのURLはカードに含めない（HPBへの自動遷移はしない）
