# バックログ

MVPリリース以降に対応する機能・改善をまとめる。
優先度は未定。実装タイミングはフェーズ計画に従う。

## フェーズ1.5

- Supabase Authの追加（メール+パスワード認証）
- マルチテナント対応（A・B美容室の別オーナー対応）
- RLSの本格稼働（フェーズ1は認証なしのため実質無効）

## フェーズ2

- ブログ投稿支援（Gemini Vision API・ヘアスタイル写真→ブログ原稿生成）
- Cloudflare R2による画像ストレージ
- 口コミ情報のDB保存（リーガルチェック・プライバシーポリシー整備後）

## 将来検討

- スタイリスト単位のアクセス権
- トーン・プロンプトのカスタマイズUI（現在はプリセット3種固定）
- Googleマップ等HPB以外の口コミ対応
- GAS → Vercel + Inngestへの移行（サロン数10超えたあたりで検討）
- Sentry・Vercel Analyticsの導入
- pg_cronによる返信文の自動削除実装
- 複数店舗対応の本格化
